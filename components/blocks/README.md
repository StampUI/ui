# Blocks

Composed UI sections — marketing, AI-native patterns, and utility blocks.  
Click any block to open the live preview and docs on stampui.com.

| Block | Description |
|---|---|
| [Animated Counter](https://stampui.com/blocks/components/animated-counter) | Number that animates to its target value with easing on mount. |
| [Announcement Bar](https://stampui.com/blocks/components/announcement-bar) | Site-wide dismissible notification banner for announcements, promotions, and alerts. |
| [CTA Banner](https://stampui.com/blocks/components/cta-banner) | Full-width call-to-action banner with headline, description, and primary/secondary action buttons. |
| [Changelog Feed](https://stampui.com/blocks/components/changelog-feed) | Vertical timeline changelog with versioned entries, date, tag badges, and feature descriptions. |
| [Cookie Consent](https://stampui.com/blocks/components/cookie-consent) | GDPR cookie consent banner with accept, decline, and customize options. Sticks to the bottom of the viewport. |
| [Empty State](https://stampui.com/blocks/components/empty-state) | Centered empty-state block with icon, title, description, and optional action button. |
| [Error Page](https://stampui.com/blocks/components/error-page) | Full-page error template for 404, 500, and other status codes with back/home navigation. |
| [FAQ Accordion](https://stampui.com/blocks/components/faq-accordion) | Animated expand/collapse FAQ list with plus/minus icons and optional section heading. |
| [Feature Comparison](https://stampui.com/blocks/components/feature-comparison) | Side-by-side plan comparison table with check/cross feature rows and highlighted recommended tier. |
| [Feature Grid](https://stampui.com/blocks/components/feature-grid) | Responsive icon + title + description feature grid. 2, 3, or 4 column layouts with optional eyebrow and headline. |
| [Gradient Text](https://stampui.com/blocks/components/gradient-text) | Text with a configurable CSS gradient fill. Supports any heading level and direction. |
| [Grid Wave](https://stampui.com/blocks/components/grid-wave) | 4×4 dot grid with diagonal ripple wave animation. |
| [Hero Section](https://stampui.com/blocks/components/hero-section) | Full-width landing page hero with headline, subtext, primary and secondary CTAs, and optional eyebrow badge. |
| [Loading Card](https://stampui.com/blocks/components/loading-card) | Card skeleton with shimmer animation for loading states. |
| [Loading Dots](https://stampui.com/blocks/components/loading-dots) | Three dots that bounce sequentially as a loading indicator. |
| [Metrics Grid](https://stampui.com/blocks/components/metrics-grid) | KPI card grid with value, label, trend direction, and percentage change indicator. |
| [Orbit Trail](https://stampui.com/blocks/components/orbit-trail) | Three dots orbiting a center point at different radii and speeds. |
| [Pricing Section](https://stampui.com/blocks/components/pricing-section) | A modern, data-driven pricing section for SaaS products. |
| [Progress Ring](https://stampui.com/blocks/components/progress-ring) | SVG circular progress ring with animated fill and percentage label. |
| [Registry Explorer](https://stampui.com/blocks/components/registry-explorer) | A searchable, filterable block registry browser with sidebar navigation. |
| [Signal Arc](https://stampui.com/blocks/components/signal-arc) | Concentric rings that expand and fade like sonar pulses. |
| [Site Footer](https://stampui.com/blocks/components/site-footer) | Multi-column site footer with brand logo, navigation links, social icons, and copyright line. |
| [Social Proof Bar](https://stampui.com/blocks/components/social-proof-bar) | Logo strip with star rating, user count, and trusted-by label for landing page credibility. |
| [Stats Strip](https://stampui.com/blocks/components/stats-strip) | Animated count-up metrics row. Strip layout (bordered, full-width) or grid layout. Numbers animate on viewport entry. |
| [Team Grid](https://stampui.com/blocks/components/team-grid) | Responsive team member card grid with avatar, name, role, bio, and social links. |
| [Testimonials Wall](https://stampui.com/blocks/components/testimonials-wall) | Masonry-style testimonial quote wall with avatar, name, role, and star rating. |
| [Typewriter Text](https://stampui.com/blocks/components/typewriter-text) | Text that types, pauses, and deletes itself in a looping typewriter effect. |
| [Usage Card](https://stampui.com/blocks/components/usage-card) | Display API usage and subscription limits. |
| [Waitlist Section](https://stampui.com/blocks/components/waitlist-section) | Email capture section for product launches with social proof count and submit feedback. |

---

Install any block with the CLI:

```bash
pnpm dlx stampui add <block>
```
