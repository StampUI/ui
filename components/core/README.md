# Core Components

Primitive UI components — inputs, overlays, layout, navigation, and feedback.  
Click any component to open the live preview and docs on stampui.com.

| Component | Description |
|---|---|
| [Accordion](https://stampui.com/blocks/components/accordion) | Vertically stacked expandable sections with smooth animation. |
| [Alert](https://stampui.com/blocks/components/alert) | Inline status message with semantic variants and icon slot. |
| [Alert Dialog](https://stampui.com/blocks/components/alert-dialog) | Confirmation dialog that interrupts the user flow for destructive actions. |
| [Animated Number](https://stampui.com/blocks/components/animated-number) | Count-up animation triggered when the element enters the viewport. Supports compact, currency, and percent formats. |
| [Aspect Ratio](https://stampui.com/blocks/components/aspect-ratio) | Maintains a consistent width-to-height ratio for media elements. |
| [Avatar](https://stampui.com/blocks/components/avatar) | User avatar with image, fallback initials, and size variants. |
| [Avatar Stack](https://stampui.com/blocks/components/avatar-stack) | Overlapping avatar images with +N overflow indicator. Supports sm/md/lg sizes. |
| [Badge](https://stampui.com/blocks/components/badge) | Compact label for statuses, counts, and categories. |
| [Breadcrumb](https://stampui.com/blocks/components/breadcrumb) | Hierarchical navigation trail with separator and ellipsis support. |
| [Button](https://stampui.com/blocks/components/button) | Versatile button with multiple variants, sizes, and loading state. |
| [Button Group](https://stampui.com/blocks/components/button-group) | Groups related buttons with collapsed borders for a unified look. |
| [Card](https://stampui.com/blocks/components/card) | Flexible card container with header, body, and footer sections. |
| [Checkbox](https://stampui.com/blocks/components/checkbox) | Accessible checkbox with indeterminate state and size variants. |
| [Collapsible](https://stampui.com/blocks/components/collapsible) | Single toggle section with open/close animation. Primitive for Accordion. |
| [Combobox](https://stampui.com/blocks/components/combobox) | Searchable single-value selector with keyboard navigation and async support. |
| [Command](https://stampui.com/blocks/components/command) | Command palette with search, groups, and keyboard shortcuts. |
| [Command Box](https://stampui.com/blocks/components/command-box) | Floating command palette trigger with shareable source display. |
| [Confirm Action](https://stampui.com/blocks/components/confirm-action) | Destructive action button that requires a second click within a timeout window. Resets automatically if you don't confirm. |
| [Context Menu](https://stampui.com/blocks/components/context-menu) | Right-click contextual menu with submenus, checkboxes, and keyboard shortcuts. |
| [Copy Button](https://stampui.com/blocks/components/copy-button) | Icon button that copies text to clipboard with animated confirmation. |
| [Date Picker](https://stampui.com/blocks/components/date-picker) | Popover date picker backed by the StampUI calendar. No external date library required. |
| [Dialog](https://stampui.com/blocks/components/dialog) | Accessible modal dialog with overlay, close button, header, and footer. |
| [Drawer](https://stampui.com/blocks/components/drawer) | Bottom sheet drawer with safe area support and smooth animation. |
| [Dropdown Menu](https://stampui.com/blocks/components/dropdown-menu) | Trigger-anchored menu with icons, submenus, checkboxes, and radio groups. |
| [Empty](https://stampui.com/blocks/components/empty) | A composable empty-state component with icon slot, title, description, and action. |
| [Field](https://stampui.com/blocks/components/field) | Composable field wrapper with label, hint, and error message for accessible forms. |
| [Hover Card](https://stampui.com/blocks/components/hover-card) | Rich preview card that appears on hover with configurable delay. |
| [Inline Edit](https://stampui.com/blocks/components/inline-edit) | Click-to-edit text with a pencil icon on hover. Supports Enter to save, Escape to cancel. |
| [Input](https://stampui.com/blocks/components/input) | Text input with size variants and full form integration. |
| [Input Group](https://stampui.com/blocks/components/input-group) | Combines an input with prefix/suffix addons for units, icons, or actions. |
| [Kbd](https://stampui.com/blocks/components/kbd) | Keyboard key display with realistic shadow and size variants. Includes KeyCombo for key sequences. |
| [Label](https://stampui.com/blocks/components/label) | Accessible form label with required indicator and disabled state. |
| [Marquee](https://stampui.com/blocks/components/marquee) | Infinite horizontal scrolling strip for logos, testimonials, or any repeating content. Pause on hover. |
| [Menubar](https://stampui.com/blocks/components/menubar) | Horizontal menu bar with dropdown submenus, labels, separators, and keyboard shortcuts. |
| [Multi Select](https://stampui.com/blocks/components/multi-select) | Multi-value selector with tag chips, overflow count, and keyboard accessibility. |
| [Native Select](https://stampui.com/blocks/components/native-select) | Styled native <select> element that respects platform conventions. |
| [Navigation Menu](https://stampui.com/blocks/components/navigation-menu) | Horizontal navigation with link items and optional dropdown content panels. |
| [Number Stepper](https://stampui.com/blocks/components/number-stepper) | Increment/decrement number input with min, max, and step. Supports controlled and uncontrolled usage. |
| [Pagination](https://stampui.com/blocks/components/pagination) | Page navigation with previous/next buttons, numbered pages, and ellipsis for large sets. |
| [Password Input](https://stampui.com/blocks/components/password-input) | Password field with show/hide toggle and an optional animated 4-bar strength meter. |
| [Popover](https://stampui.com/blocks/components/popover) | Floating panel anchored to a trigger with focus management and Escape-to-close. |
| [Progress](https://stampui.com/blocks/components/progress) | Progress bar with size and semantic variant support. |
| [Radio Group](https://stampui.com/blocks/components/radio-group) | Keyboard-navigable radio button group with size variants. |
| [Resizable](https://stampui.com/blocks/components/resizable) | Drag-to-resize panel layout with horizontal and vertical split support. |
| [Scroll Area](https://stampui.com/blocks/components/scroll-area) | Custom-styled scrollable container with consistent cross-browser scrollbars. |
| [Select](https://stampui.com/blocks/components/select) | Accessible single-value select with groups, search, and custom trigger. |
| [Separator](https://stampui.com/blocks/components/separator) | Horizontal or vertical divider line with accessible semantics. |
| [Sheet](https://stampui.com/blocks/components/sheet) | Slide-in panel from any edge (top, right, bottom, left). |
| [Sidebar](https://stampui.com/blocks/components/sidebar) | Vertical navigation sidebar with collapsible groups, icons, badges, and collapsed mode. |
| [Skeleton](https://stampui.com/blocks/components/skeleton) | Animated placeholder for loading states with size variants. |
| [Slider](https://stampui.com/blocks/components/slider) | Range input with track, thumb, and step support. Keyboard accessible. |
| [Sonner](https://stampui.com/blocks/components/sonner) | Toast notification system with dark mode and queue support. |
| [Spinner](https://stampui.com/blocks/components/spinner) | SVG loading spinner with size and color variants. |
| [Status Pulse](https://stampui.com/blocks/components/status-pulse) | Animated ping dot for live status indicators. States: online, offline, processing, error, warning. |
| [Switch](https://stampui.com/blocks/components/switch) | Toggle switch for boolean settings with size and color variants. |
| [Table](https://stampui.com/blocks/components/table) | Semantic HTML table with styled header, body, and footer sections. |
| [Tabs](https://stampui.com/blocks/components/tabs) | Layered sections of content controlled by a tab list. Full keyboard navigation. |
| [Tag Input](https://stampui.com/blocks/components/tag-input) | Multi-tag text input. Press Enter or comma to add tags, Backspace to remove the last one. |
| [Textarea](https://stampui.com/blocks/components/textarea) | Multi-line text input with auto-resize and size variants. |
| [Toggle](https://stampui.com/blocks/components/toggle) | Two-state button with pressed/unpressed states and three variants. |
| [Toggle Group](https://stampui.com/blocks/components/toggle-group) | Connected toggle buttons for single or multiple selection. |
| [Tooltip](https://stampui.com/blocks/components/tooltip) | Accessible label popup triggered by hover or focus. |
| [Typing Indicator](https://stampui.com/blocks/components/typing-indicator) | Three bouncing dots for AI or chat typing state. Bubble and bare variants. |

---

Install any component with the CLI:

```bash
pnpm dlx stampui add <component>
```
